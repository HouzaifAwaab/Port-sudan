## المهن والخدمات

- بناء منزل كامل
- بناء جزء للمنزل
- تركيب سراميك
- عمل بلاط

- نقاشة جزء للمنزل

### ⚡ كهربائي

- تركيب مكيفات

- غسالات
- شاشات

- حداد

### 🚚 توصيل ونقل
#### توصيل مشوار:
- ركشه ٣نفر
- امجاد ٧نفر
- حافله ٢٤نفر

#### توصيل رحله:
- حافله ٢٤نفر (للأجرة بالكيلو)
- حافله ٢٤نفر (للأجرة بالرقبة)
- حافله ٢٤نفر (للعروس)

#### نقل:
- تكتوك (وزن حتى ٧٠٠ كيلو)
- دفار (وزن طن فأكثر)
